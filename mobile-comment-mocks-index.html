Mobile Comment Section Mocks

Full mobile experience with collapsed/expanded states

Each mock demonstrates a complete mobile comment section with an input that starts small and expands when the user interacts with it. Click any card to view the interactive prototype.

1
Smooth Expand
In-place growth with seamless transitions
A fluid, in-place expansion where the input grows vertically and reveals actions smoothly. The input stays in the same location, creating a natural, non-disruptive experience. Perfect for users who want to stay in context.
Key Features
Single-line collapsed state
Smooth vertical expansion
Actions slide in from bottom
Avatar grows slightly on expand
Click outside to collapse
Stays in fixed position
2
Slide Up Modal
Focused writing experience
A modal-based approach where tapping the input bar opens a dedicated composing interface that slides up from the bottom. This provides maximum focus and screen real estate for writing longer comments.
Key Features
Minimal collapsed trigger
Full-screen modal experience
Rounded top corners
Drag handle for dismissal
Backdrop overlay
User info in header
3
Minimal Grow
Progressive disclosure with quick actions
A smart, adaptive input that grows with your content. Shows a quick-send button for short messages and reveals full actions for longer compositions. Balances simplicity with power.
Key Features
Compact initial state
Auto-height textarea growth
Quick send for short replies
Extended actions on focus
Smooth transitions
Progressive disclosure
When to Use Each Approach

Smooth Expand

Best for contexts where users need to stay aware of surrounding content. Ideal for comment threads where context matters.

Slide Up Modal

Perfect for longer-form comments or when you want to give users maximum focus. Great for thoughtful, detailed responses.

Minimal Grow

Excellent for quick interactions with the option for more. Adapts to user intent—simple for quick replies, powerful for detailed comments.
